# Instrucciones para uso de este proyecto.

## En la terminal:
- git clone "git url del repo"
- python3 -m venv venv
- source vemv/bin/activate
- Para desactivar el entorno (deactivate)

## En cuanto a Docker:
    - docker-compose build
    - docker-compose up -d
    - docker-compose ps
    - docker-compose exec app-python-practice bash
